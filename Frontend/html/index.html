About Pharmly

We're building the future of intelligent, compliant, and human-centered pharmacy management — powered by artificial intelligence and driven by an unwavering commitment to patient safety.

Our Story

Pharmly was born from a simple observation: pharmacy management in the modern world was still stuck in outdated systems — paper prescriptions, manual inventory tracking, and zero real-time insights. We saw an opportunity to change that fundamentally.

Founded by a team of healthcare technologists, AI researchers, and pharmacists, Pharmly set out to build a platform that doesn't just digitize existing workflows — it reimagines them entirely. We asked ourselves: what would pharmacy management look like if it were designed from the ground up today, with everything we know about AI, cloud computing, and user experience?

The result is a platform that brings together natural language AI (so pharmacists and patients can just talk to it), real-time inventory intelligence, prescription compliance enforcement, and a beautiful dashboard that makes complex data feel simple and actionable.

Today, we serve hundreds of pharmacies across the country, processing thousands of orders daily, catching prescription violations before they happen, and helping pharmacy teams work smarter — not harder.

But what we're most proud of isn't the technology. It's the patient safety outcomes. Every prescription block, every low-stock alert, every dosage warning caught by our compliance engine represents a real person being protected. That's why we built this.

Join us in building a healthier world — one prescription at a time.

🛡️ Safety & Precautions

At Pharmly, the health and safety of every patient is our absolute highest priority. This section provides comprehensive, plain-language guidance on safe medication use, prescription compliance, allergy awareness, data privacy, and platform security.

🤖 AI & Chat
POST/chat-agent

Send a message to the AI Pharmacy Assistant. Backend handles NLU, medication lookup, prescription checking.

Request: { "message": "I need paracetamol 10 tablets" }
Response OK: { "reply": "Got it! Processing...", "status": "approved" }
Response Rx: { "status": "rejected", "reason": "prescription_required" }
POST/voice-agent

Voice-to-AI endpoint via Vapi SDK. Processes voice input and returns AI response.

Vapi SDK: const vapi = new Vapi("PUBLIC_API_KEY")
Button: vapi.start("ASSISTANT_ID")
📄 Prescription Verification
POST/upload-prescription

Upload and verify a patient prescription. Backend validates against the requested medicine.

Request: { "customer_id": "USR001", "medicine": "Amoxicillin 500mg" }
Response OK: { "status": "verified", "verification_id": "RX-4821" }
📦 Orders
GET/orders

Get all orders. Params: ?status=completed&limit=50&page=1

POST/orders

Create a new order. Prescription check enforced server-side.

{ "customer_id": "USR001", "medicine": "Paracetamol 650", "quantity": 10 }
PUT/orders/:id

Update an order status or details.

DELETE/orders/:id

Cancel and delete an order (admin only).

💊 Inventory / Medicines
GET/inventory

Get all medicines with stock. Params: ?category=antibiotic&low_stock=true

POST/inventory

Add a new medicine.

{ "name": "Amoxicillin 500mg", "stock": 50, "restricted": true }
GET/inventory/alerts

Get low-stock and critical-stock alerts for notification bell.

👥 Customers
GET/customers

Get all registered customers.

POST/customers

Register a new customer.

{ "name": "John Doe", "email": "john@example.com", "phone": "+1-555-0100" }
📊 Analytics & Dashboard
GET/analytics/revenue

Revenue data. Params: ?period=week|month|today

GET/analytics/top-sellers

Top selling medicines for given period.

GET/analytics/dashboard-summary

All dashboard metrics: revenue, orders, low-stock count, prescription blocks.

🔐 Authentication
POST/auth/loginPublic

Login with email/password. Returns JWT token.

Request: { "email": "admin@pharmly.com", "password": "..." }
Response: { "token": "eyJhbGc..." }
POST/auth/logout

Invalidate the current session token.
